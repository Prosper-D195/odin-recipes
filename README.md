# odin-recipes 
Recettes Béninoises.
Ce projet est un site web qui présente des recettes traditionnelles béninoises. Le Bénin est un pays de l'Afrique de l'Ouest connu pour sa riche cuisine variée et délicieuse. Ce site web vise à partager ces recettes dans le monde entier.
Reccettes Présentées
* Akassa: C'est une sorte de pâte faite à base de maïs cuit, moulu et trempé.
* Igname pilée: C'est une pâte gluante obtenue traditionnellement en pilant et en pétrissant de l'igname bouillie.
* Le djongoli: C'est une purée de haricot à l'huile de palme et à la farine de maïs.
Compétences démontrées:
Création d'un site web reponsive pour présenter des recettes de cuisine.
* Utilisation de HTML pour structurer le contenu.
* Utilisation de CSS pour styler le site web.
* Utilisation de JavaScript pour ajouter des fonctionnalités interatives.
